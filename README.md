## Safe and Ethical Use

All technical material is intended for lawful, authorized, and ethical use. Use security tools and testing procedures only when you own the environment, have explicit written permission, are working in an approved laboratory, or are operating within a documented professional scope.

> **Technical ability does not create permission.**

Do not use these materials to access, test, disrupt, monitor, alter, or collect information from systems, applications, AI models, accounts, networks, or data without authorization.
